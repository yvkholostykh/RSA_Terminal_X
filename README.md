# RSA_Terminal_X
Public key cryptosystem, RSA, IMPLEMENTED AN ATTACK (Cube root)
RSA Crypto Terminal
🔐 Интерактивная реализация алгоритма RSA с поддержкой файлов и визуальным сопровождением через эмодзи. Идеальна для обучения криптографии и экспериментов с шифрованием.

!
!

✨ Особенности
Интерактивный режим — пошаговое выполнение RSA с пояснениями.

Поддержка файлов — шифрование и расшифровка текстовых файлов любого размера.

Генерация ключей с выбором размера (от 128 до 4096 бит) или ручным вводом простых чисел.

Визуальное сопровождение через эмодзи для интуитивного понимания этапов.

Демонстрация атак — пример атаки Cube Root при малой экспоненте e=3.

Сохранение данных — все ключи и сообщения автоматически сохраняются в папке скрипта.

Проверка простоты через тест Миллера‑Рабина.

Полная локализация на русском языке.

🔷 Ваш выбор:
Основные сценарии
1. Генерация ключей (2)
Выберите размер простых чисел (рекомендуется 512–2048 бит).

Либо введите p и q вручную.

Выберите экспоненту e (по умолчанию 65 537).

Ключи сохранятся в public_key.txt и private_key.txt.

2. Шифрование файла (4)
Убедитесь, что загружен открытый ключ (3) или сгенерирован новый (2).

Укажите имя файла с открытым текстом (по умолчанию message.txt).

Введите имя для файла шифртекста (по умолчанию encrypted.txt).

3. Расшифровка файла (5)
Загрузите закрытый ключ (3).

Укажите имя файла с шифртекстом (по умолчанию encrypted.txt).

Введите имя для расшифрованного файла (по умолчанию decrypted.txt).

4. Интерактивный режим (1)
Пошагово выполните RSA: ввод p, q, e, сообщение m, шифрование c = m^e mod n, расшифровка m' = c^d mod n.

5. Демонстрация атаки (8)
Введите шифртекст c и экспоненту e (обычно 3), чтобы попробовать восстановить исходное сообщение через извлечение корня.

🗂️ Структура файлов
rsa_terminal.py — основной скрипт.

public_key.txt — открытый ключ (e, n).

private_key.txt — закрытый ключ (d, n).

message.txt — исходный текст для шифрования.

encrypted.txt — шифртекст (числа, разделённые переносами строк).

decrypted.txt — расшифрованный текст.

📚 Как это работает
Программа реализует классический алгоритм RSA:

Генерация ключей:

Выбираются два больших простых числа p и q.

Вычисляется модуль n=p⋅q и функция Эйлера φ(n)=(p−1)(q−1).

Выбирается открытая экспонента e, взаимно простая с φ(n).

Находится закрытая экспонента d=e 
−1
 modφ(n).

Шифрование: c=m 
e
 modn.

Расшифрование: m=c 
d
 modn.

Для проверки простоты используется тест Миллера‑Рабина. Преобразование текста в число выполняется блоками, исходя из размера модуля.
